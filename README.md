# 🎮 Game Machine

Game Machine é uma aplicação web interativa que reúne Jogo da Velha (Tic-Tac-Toe) e Pedra-Papel-Tesoura (Jo-ken-po) em uma experiência divertida, responsiva e acessível.

O código do projeto está escrito em inglês para padronização e facilidade de leitura por outros desenvolvedores, mas toda a interface e conteúdo para o jogador estão em português, garantindo que a experiência seja natural e divertida.

# 🔗 Link para o Projeto

A aplicação está hospedada em <link>

# ✨ Funcionalidades

🎯 Jogo da Velha contra o computador.

✋✌✊ Pedra-Papel-Tesoura contra o computador.

🏆 Mensagens dinâmicas indicando vitória, derrota ou empate.

🔄 Botão para reiniciar o jogo a qualquer momento.

📱 Layout totalmente responsivo, funcionando bem em desktop e mobile.

♻️ Componentes reutilizáveis, facilitando manutenção e expansão futura.

💡 Uso de HTML semântico, melhorando acessibilidade e SEO.

# 🛠 Tecnologias Utilizadas

React – construção de interfaces dinâmicas e reativas.

TypeScript – tipagem estática que garante segurança e previsibilidade.

CSS Modules – estilos isolados por componente, evitando conflitos.

Flexbox e Grid – alinhamento e layout responsivo.

HTML semântico – uso correto de <header>, <main>, <section>, <h1>/<h2> e <button>.

# 💡 Boas Práticas Aplicadas

Componentização: cada elemento da interface é um componente isolado e reutilizável.

Separação de lógica e apresentação: o código do jogo é independente da UI, facilitando manutenção.

Acessibilidade: botões com aria-labels, textos semânticos e contraste adequado.

Responsividade: adapta-se a diferentes tamanhos de tela e dispositivos.

Código em inglês + conteúdo em português: facilita entendimento por desenvolvedores, enquanto mantém a experiência do jogador em português.

Animações e efeitos visuais: mensagens e botões com efeitos de glow, hover e transições suaves.

# 🚀 Como Executar o Projeto

1. Clone o repositório:

git clone https://github.com/seu-usuario/game-machine.git

2. Instale as dependências:

npm install

3. Inicie o servidor de desenvolvimento:

npm start

4. Abra no navegador:

http://localhost:5173
